## cmake

	$ sudo dnf install glibc-devel
	
	$ sudo dnf install cmake
	
	$ wget <cmake download link>

	$ tar -xvzf <cmake-xxx.tar.gz>

	$ cd <cmake directory>
	
	$ ./configure

	$ make

	$ make install
